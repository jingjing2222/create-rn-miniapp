# Display Patterns

- `amount-top`: 금액 hero, 송금/결제/잔액 상단 요약
- `asset`: 이미지/아이콘/Lottie를 frame 안에 넣어야 하는 surface
- `badge`: 상태 chip, 강조 라벨
- `bottom-info`: 하단 안내, 면책, 법적 고지
- `carousel`: 가로 스와이프 카드/배너
- `chart`: 막대 차트, docs slug alias는 `Chart/bar-chart`
- `gradient`: gradient visual utility
- `highlight`: 온보딩/튜토리얼 강조
- `post`: 공지, 이벤트, 장문 본문
- `shadow`: shadow visual utility

## quick comparisons

- `bottom-info`는 짧은 하단 고지다.
- `post`는 서술형 본문이다.
- `asset`은 frame이 필요한 미디어다.
- `badge`는 상태 인식용 label이다.
